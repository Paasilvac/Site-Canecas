# 🛍️ Site de Canecas Geek

Landing page desenvolvida como projeto de estudo para simular uma loja virtual de produtos geek. O projeto tem como objetivo aplicar conceitos de desenvolvimento web utilizando HTML, CSS e JavaScript, oferecendo uma experiência simples de navegação e compra por meio do WhatsApp.

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## 💡 Funcionalidades

* Exibição de produtos em catálogo
* Carrinho de compras
* Cálculo automático do valor total da compra
* Alteração de quantidade de itens
* Finalização do pedido via WhatsApp
* Layout responsivo para diferentes dispositivos

## 🎯 Objetivos do projeto

* Praticar estruturação de páginas web
* Aplicar manipulação do DOM com JavaScript
* Desenvolver uma interface intuitiva para o usuário
* Simular um fluxo básico de compra em um e-commerce

## 📚 Aprendizados

Durante o desenvolvimento deste projeto foram aplicados conhecimentos de:

* Estruturação semântica com HTML
* Estilização utilizando CSS
* Manipulação de elementos com JavaScript
* Organização do código
* Lógica de programação
